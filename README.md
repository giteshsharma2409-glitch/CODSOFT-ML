# Customer Churn Prediction

## Overview
This project predicts whether a customer is likely to leave a subscription-based service using historical customer data and machine learning algorithms.

## Features
- Data preprocessing
- Exploratory Data Analysis
- Feature Engineering
- Customer churn prediction
- Model evaluation and comparison

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Machine Learning Models
- Logistic Regression
- Random Forest
- Gradient Boosting
- XGBoost

## Dataset
Customer Churn Dataset

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC Score

## Results
Identified customers at risk of churn, helping businesses improve customer retention strategies.

## Future Improvements
- Real-time churn monitoring
- Deep Learning implementation
- Business dashboard integration

## Features

- TF-IDF feature extraction
- Genre prediction from movie descriptions
- Model comparison and evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK

## Machine Learning Models
- Logistic Regression
- Naive Bayes
- Support Vector Machine (SVM)

## Dataset
Movie Genre Classification Dataset

## Evaluation Metrics
- Accuracy Score
- Classification Report
- Confusion Matrix

## Results
Successfully classified movies into their respective genres based on textual information.

# Spam SMS Detection

## Overview
This project uses Machine Learning and Natural Language Processing (NLP) techniques to classify SMS messages as Spam or Ham (Legitimate). The model analyzes text messages and predicts whether they are unwanted spam messages.

## Features
- Text preprocessing and cleaning
- TF-IDF Vectorization
- Multiple ML algorithms comparison
- Cross-validation
- User input prediction

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
- Matplotlib
- Seaborn

## Machine Learning Models
- Multinomial Naive Bayes
- Logistic Regression
- Random Forest Classifier
- Support Vector Machine (SVM)

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

## Dataset
SMS Spam Collection Dataset

## Results
Achieved high accuracy in identifying spam and legitimate messages.

## Future Improvements
- Deep Learning implementation
- Real-time SMS filtering
- Deployment using Flask/Streamlit
